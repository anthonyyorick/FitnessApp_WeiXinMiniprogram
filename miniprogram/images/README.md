# 图片资源目录

此目录用于存放小程序所需的图片资源，包括：

## 必需图片
- `home.png` - 首页图标（未选中状态）
- `home-active.png` - 首页图标（选中状态）
- `sport.png` - 运动图标（未选中状态）
- `sport-active.png` - 运动图标（选中状态）
- `health.png` - 健康图标（未选中状态）
- `health-active.png` - 健康图标（选中状态）
- `my.png` - 我的图标（未选中状态）
- `my-active.png` - 我的图标（选中状态）
- `default-avatar.png` - 默认头像

## 课程图片
- `course1.jpg` - 课程1封面
- `course2.jpg` - 课程2封面
- `course3.jpg` - 课程3封面

## 说明
1. 底部导航图标建议尺寸：81px × 81px
2. 头像图片建议尺寸：120px × 120px
3. 课程封面建议尺寸：320px × 200px
4. 所有图片建议使用PNG或JPG格式

## 临时替代方案
如果暂时没有图片资源，可以使用以下方式：
1. 使用在线图片服务（如：https://via.placeholder.com/）
2. 使用emoji作为临时图标
3. 使用纯色背景作为占位符
