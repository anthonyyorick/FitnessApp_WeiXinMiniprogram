# 健康页面图表测试说明

## 测试目的
验证健康页面的趋势图表功能是否正常工作。

## 测试步骤

### 1. 基础功能测试
1. 打开小程序，进入健康页面
2. 检查页面是否正常加载
3. 验证图表容器是否显示

### 2. 图表渲染测试
1. 检查趋势图区域是否有图表显示
2. 验证图表是否包含数据线条
3. 检查图表标题是否正确显示

### 3. 健康指标切换测试
1. 点击"心率"标签
   - 验证图表显示心率数据
   - 检查Y轴标签为"心率 (bpm)"
   - 验证数据范围在60-90之间

2. 点击"血压"标签
   - 验证图表显示两条线（收缩压和舒张压）
   - 检查Y轴标签为"血压 (mmHg)"
   - 验证图例显示"收缩压"和"舒张压"

3. 点击"睡眠"标签
   - 验证图表显示睡眠时长数据
   - 检查Y轴标签为"睡眠时长 (小时)"
   - 验证数据范围在5-9之间

4. 点击"体重"标签
   - 验证图表显示体重数据
   - 检查Y轴标签为"体重 (kg)"
   - 验证数据范围在64-68之间

### 4. 时间范围切换测试
1. 点击"近7天"
   - 验证图表显示7个数据点
   - 检查X轴标签格式为"M/D"

2. 点击"近30天"
   - 验证图表显示30个数据点
   - 检查数据密度是否合适

3. 点击"近一年"
   - 验证图表显示365个数据点
   - 检查图表是否正常渲染

### 5. 交互功能测试
1. 触摸图表区域
   - 验证tooltip是否显示
   - 检查tooltip内容是否正确

2. 切换健康指标和时间范围
   - 验证图表是否实时更新
   - 检查数据是否正确切换

## 预期结果

### ✅ 正常情况
- 图表正常显示，包含数据线条
- 切换健康指标时图表正确更新
- 切换时间范围时数据点数量正确
- tooltip功能正常工作
- 图表样式美观，颜色搭配合理

### ❌ 异常情况
- 图表区域空白或显示错误
- 切换功能不生效
- 数据范围异常
- 图表样式错乱

## 故障排除

### 问题1：图表不显示
**可能原因**：
- ECharts组件未正确引入
- 图表初始化失败
- 数据未正确加载

**解决方案**：
1. 检查控制台是否有错误信息
2. 确认ECharts组件路径正确
3. 检查数据加载逻辑

### 问题2：图表显示异常
**可能原因**：
- 数据格式错误
- 图表配置问题
- 样式冲突

**解决方案**：
1. 检查数据格式是否正确
2. 验证图表配置参数
3. 检查CSS样式是否冲突

### 问题3：切换功能不生效
**可能原因**：
- 事件绑定错误
- 数据更新逻辑问题
- 图表更新方法未调用

**解决方案**：
1. 检查事件绑定是否正确
2. 验证数据更新逻辑
3. 确认图表更新方法被调用

## 技术说明

### 图表组件
- 使用ECharts微信小程序版本
- 支持新旧两种Canvas API
- 自动适配不同屏幕尺寸

### 数据生成
- 使用模拟数据确保图表正常显示
- 支持不同时间范围的数据生成
- 根据健康指标类型生成相应数据

### 图表配置
- 响应式设计，适配不同屏幕
- 美观的配色方案
- 支持tooltip和交互功能
