# MyFitnessApp - 健身小程序

一个功能完整的微信小程序健身应用，包含运动记录、健康监测、课程推荐等功能。

## 功能特性

### 🏃‍♂️ 运动功能
- **运动记录**: 支持跑步、步行、骑行等多种运动类型
- **数据统计**: 步数、距离、时长、卡路里消耗统计
- **时间筛选**: 今日、本周、本月数据查看
- **运动历史**: 详细的运动记录列表

### 💪 健康监测
- **心率监测**: 实时心率数据记录
- **血压记录**: 血压数据管理
- **睡眠质量**: 睡眠时长和质量分析
- **体重管理**: 体重变化趋势跟踪

### 📚 课程系统
- **健身课程**: 多种类型的健身课程
- **难度分级**: 初级、中级、高级课程分类
- **课程详情**: 详细的课程介绍和指导

### 👤 个人中心
- **用户信息**: 个人资料管理
- **设置管理**: 通知、单位、隐私设置

## 技术架构

### 前端技术栈
- **微信小程序**: 原生小程序开发
- **WXML/WXSS**: 页面结构和样式
- **JavaScript**: 业务逻辑处理
- **ECharts**: 数据可视化图表

### 后端技术栈
- **Node.js**: 服务器运行环境
- **Express**: Web应用框架
- **MongoDB**: 数据库存储
- **JWT**: 用户认证

## 开发说明

### 默认数据使用
1. 应用启动时会自动加载默认数据
2. 在没有网络连接或后端服务时，使用默认数据展示
3. 用户登录后，会尝试从后端获取真实数据
4. 网络请求失败时，自动回退到默认数据

### 数据切换逻辑
- **未登录状态**: 始终使用默认数据
- **已登录状态**: 优先使用后端数据，失败时使用默认数据
- **网络异常**: 自动使用默认数据保证用户体验

## 安装和运行

### 小程序端
1. 使用微信开发者工具打开 `miniprogram` 目录
2. 配置小程序AppID
3. 编译运行

### 后端服务
1. 进入 `server` 目录
2. 安装依赖: `npm install`
3. 启动服务: `npm start`
